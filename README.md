## 🧩 What I Do

- 🚀 Build infrastructure where failure is not an option and recovery is not guaranteed
- 🖥️ Architect and manage **virtualized environments (Proxmox, LXC, VMs)**
- 🐳 Build and orchestrate **containerized stacks (Docker, Compose, self-hosted services)**
- 🌐 Design **segmented networks (VLANs, UniFi, security-first approach)**
- ⚙️ Automate workflows using **APIs, and custom integrations**
- 📊 Improve IT operations with **data-driven decision making******
- 🛰️ Support mission-critical systems with constraints similar to off-world environments 
- 🚀 Occasionally build systems that operate far outside the data center
